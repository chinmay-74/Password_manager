# 🔐 Password Manager (Tkinter GUI)

A simple and secure Password Manager built using Python and Tkinter.
This application allows users to generate, store, and retrieve passwords easily.

---

## ✨ Features

* 🔑 Generate strong random passwords
* 📋 Copy password to clipboard automatically
* 💾 Save credentials (Website, Email, Password) in a JSON file
* 🔍 Search for saved passwords by website
* ⚠️ Error handling for missing files and empty fields
* 🖥️ Simple and clean GUI using Tkinter

---

## 🛠️ Tech Stack

* Python 🐍
* Tkinter (GUI)
* JSON (Data storage)
* Pyperclip (Clipboard handling)
* Random module

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/chinmay-74/Password_manager.git
   ```

2. Navigate to the project folder:

   ```bash
   cd Password_manager
   ```

3. Install required library:

   ```bash
   pip install pyperclip
   ```

4. Run the application:

   ```bash
   python Passwordmanager.py
   ```

* `Passwordmanager.py` → Main application file
